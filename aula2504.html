Rede de Computadores

Uma rede de computadores é um conjunto de equipamentos interligados de maneiras a trocarem informações e compartilharem recursos, como arquivos de dados gravados, impressoras, modems, softwares e outros equipamentos.

Importância das Redes de Computadores

Proporcionam economia, compartilhamento de recursos, como de impressoras e projetores, bem como de serviços, que é o caso da internet. Permite também, por meio do compartilhamento de arquivos, o trabalho em equipes dos colaboradores.

LAN WAN

LAN (local area network) é uma rede de computadores que cobre uma área limitade, geralmente um edifício ou um campus. É usada para conectar em um espaço físico limitado, como uma casa, escritório ou sala de aula. Já WAN (wide area network) é uma rede de computadores que abrange uma área geográfica maior, geralmante uma cidade, um estado ou até mesmo um país. Ela é usada para conectar dispositivos que estão fisicamente distantes de si.
