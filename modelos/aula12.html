HTML (HyperText Markup Language) é uma linguagem de marcação que define a estrutura e o conteúdo de uma página web. É a linguagem base que os navegadores usam para interpretar e exibir páginas da internet. Define o significado e a organização dos elementos da página, como textos, imagens, links e tabelas. Em resumo: HTML é o código que constrói a estrutura básica de uma página web, indicando ao navegador como os elementos devem ser exibidos.
